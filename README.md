# LLM Comparison: Document Summarization

This repository contains the benchmarking and evaluation framework used to compare the performance of leading Large Language Models (LLMs) on a technical document summarization task. 

The goal of this project is to analyze how different models handle dense technical data, complex metrics, and strict prompting constraints.
